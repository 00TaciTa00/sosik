// DETECTION & AI PIPELINE
TRIGGER
웹훅 수신 / 수동 체크
웹훅 ON — 레포별 고유 URL로 신호 수신 시 자동 실행
localhost:{포트}/webhook/{레포-ID}
수동 — "AI 요약 생성" 버튼으로 언제든 실행 가능 (웹훅 ON/OFF 무관)
수동 실행 + 기존 노트 있으면 → 덮어쓰기 확인 후 재생성
PROCESS
레포별 새 커밋 동시 감지
등록된 모든 레포 병렬 체크
마지막 확인 SHA 이후 변경사항 탐색
GitLab/GitHub API 또는 로컬 git 폴더 (레포별 선택)
NOTIFY
토스트 알림
새 배포 감지 시 앱 내 토스트 알림
레포 이름 + 커밋 정보 함께 표시
사이드바 레포에 알림 뱃지 표시
PROCESS
git diff 추출
감시 브랜치: origin/main 고정 (브랜치 지정은 차후 고도화)
범위: 이전 릴리즈 노트 마지막 SHA → 현재 HEAD (복수 커밋 → 릴리즈 노트 1개)
레포별 독립 보안 제외 규칙 적용 후 추출 — 순서 고정
AI
AI 요약 생성
Claude / GPT (전역 설정, 교체 가능)
요약 언어는 레포별 설정 — 한국어 / 영어 / 둘 다 (토큰 2배 경고)
요약 스타일은 레포별 설정 — 상세 / 간결 / 기술적
변경 유형 자동 분류: 버그수정 / 기능추가 / UI변경 / 성능개선
STORE
SQLite 로컬 저장
레포별 초안 기록 영구 저장
레포별 설정값 + 전역 설정값 저장
API Key는 electron-store 암호화 저장
