# Sosik — 업데이트 노트 자동 생성 데스크톱 앱 기획서 v4

> 작성일: 2026-04-06

---

## 전체 플로우

```bash
앱 실행
→ 등록된 레포지토리 목록 로드
→ 각 레포별 새 커밋 동시 체크
→ 새 커밋 감지된 레포에 토스트 알림
→ 레포 선택
→ git diff 추출 (레포별 보안 파일 제외 규칙 적용)
→ AI 요약 생성 (레포별 언어 설정 적용)
→ 해당 레포의 초안 확인 및 수정
→ 포맷 선택 후 발행
→ SQLite에 레포별 기록 저장
```

---

## 모듈별 확정 사항

### 1. 레포지토리 관리

- 레포 **추가 / 수정 / 삭제** 가능
- 레포별 개별 설정
  - GitLab 또는 GitHub 선택 (최초 등록 시에만 설정, 이후 읽기 전용)
  - API 방식 또는 로컬 git 폴더 방식 선택 (최초 등록 시에만 설정, 이후 읽기 전용)
  - 저장소 URL (최초 등록 시에만 설정, 이후 읽기 전용)
  - 액세스 토큰 (언제든 수정 가능)
  - 보안 파일 제외 규칙 (레포별 완전 독립)
  - AI 요약 언어 (레포별 독립 설정)
- 레포 목록은 사이드바에서 빠르게 전환

### 2. 배포 감지

- **웹훅 수신 방식**: GitLab/GitHub/로컬 레포에서 Sosik 앱으로 직접 신호 전송
  - ON 시 로컬 HTTP 서버를 열어 웹훅 수신 대기
  - GitLab/GitHub: 레포 웹훅 설정에 로컬 URL 등록
  - 로컬 git 폴더: post-receive git hook으로 신호 전송
  - OFF 시 수동 체크만 가능, 웹훅 관련 설정란 비활성화
- **AI 요약 생성 트리거**
  - 웹훅 ON: 신호 수신 시 자동으로 diff 추출 → AI 요약 생성
  - 웹훅 OFF: 수동 "AI 요약 생성" 버튼으로만 실행
  - 웹훅 ON 상태에서도 수동 실행 가능
  - 수동 실행 시 기존 릴리즈 노트가 있으면 최신 diff로 새로 생성하여 덮어씀 (덮어쓰기 확인 필요)
- **수동 체크 버튼** (전체 / 특정 레포 선택) — 웹훅 ON/OFF 무관하게 항상 사용 가능
- 새 커밋 감지 시 OS 데스크톱 알림 (화면 모서리 표시, 소리 없음)
- 시작 프로그램 등록 옵션 (설정에서 켜기/끄기)

### 3. diff 소스 (레포별 설정)

| 방식 | 설명 |
|------|------|
| GitLab/GitHub API | 원격 저장소에서 직접 가져옴 |
| 로컬 git 폴더 | PC에 clone된 폴더 경로 지정 |

### 4. 보안 파일 제외 (레포별 완전 독립)

- 전역 규칙 없음
- 레포마다 완전히 독립된 규칙 설정
- 경로 패턴 예시: `**/.env`, `**/secrets/**`
- 확장자 패턴 예시: `.pem`, `.key`
- 레포 추가 및 레포 설정 화면에서 관리

### 5. AI 요약

- AI 제공자: 레포별 개별 설정 (Claude / GPT)
- API Key: 전역 설정 (교체 가능)
- **요약 언어: 레포별 개별 설정**
  - 한국어만
  - 영어만
  - 한국어 + 영어 동시 (토큰 약 2배 소모 — 선택 시 경고 안내)
- **요약 스타일: 레포별 개별 설정**
  - 상세
  - 간결
  - 기술적
- 변경 유형 자동 분류: 버그수정 / 기능추가 / UI변경 / 성능개선

### 6. 로컬 저장 (SQLite)

| 저장 항목 | 범위 |
| ------ | ------ |
| 레포지토리 목록 및 설정 | 레포별 |
| 보안 파일 제외 규칙 | 레포별 |
| 초안 생성 기록 | 레포별 |
| AI 제공자 | 레포별 |
| API Key | 전역 |
| 웹훅 수신 설정 (ON/OFF, 포트, 시크릿 토큰) | 전역 |
| 앱 설정 (언어, 테마, 시작 프로그램 등록) | 전역 |

### 7. 앱 UI

#### 사이드바

- 접기 / 펼치기 가능
  - **펼침 상태**: 레포 이름 + 새 커밋 알림 뱃지 표시
  - **접힘 상태**: 레포 이름 자동 이니셜 아이콘 + 알림 뱃지만 표시
- 구성 (위 → 아래)
  1. **[+ 레포 추가]** 버튼 (최상단)
  2. 레포 목록 (클릭 시 해당 레포 선택)
  3. **[설정]** 아이콘 (하단 고정) — 전역 설정 진입 (AI 제공자/API Key, 테마, 언어, 시작 프로그램 등)

#### 메인 영역 (레포 선택 시)

상단 탭으로 아래 4개 페이지를 전환:

| 탭 | 주요 내용 |
|----|-----------|
| 대시보드 | 레포 상태 요약, 새 커밋 새로고침 버튼 |
| 릴리즈 노트 목록 | AI 요약 초안 확인 및 수정, 발행, 새로고침 버튼 |
| 깃 그래프 | 커밋 히스토리 시각화, 새로고침 버튼 |
| 설정 | 레포별 설정 (보안 제외 규칙, 요약 언어, AI 제공자, 액세스 토큰 수정), 레포 삭제 |

> 새로고침 버튼은 **대시보드 / 릴리즈 노트 목록 / 깃 그래프** 각 페이지 콘텐츠 안에 배치 (설정 탭 제외)

#### 공통

- 라이트 / 다크모드 전환
- 연록 ~ 초록 ~ 청록 사이의 컬러 (햇살이 들어오는 숲)
- 한 / 영 언어 전환
